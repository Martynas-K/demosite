## DemoSite web app
A website design using React.
	
## Technologies
Project is created with:
* React v16.12.0
* Redux
* node-sass-chokidar
* Axios
* FontAwesome
* Meteo.lt API
	
## Setup
To run this project, install it locally using npm.
To get the frontend running locally:

* Clone this repo 
* Run `npm install` to install all required dependencies
* Run `npm run start` to start the local server (this project uses create-react-app)
* Run `npm run watch-css` watch for changes in the file system
